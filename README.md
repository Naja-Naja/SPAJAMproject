# Unity2Dtemplate_2021
Unity2Dのテンプレートです。  
バージョンは2021.3.2f1を使用しています
## 含まれているアセットや事前準備内容
### 準備
- URP対応
- 簡単なフォルダ分け
- AudioMixerの初期設定
- AssemblyDefinitionの有効化
- ScriptableObjectの定義
- シーンテンプレートの作成
  - 簡単なBGM/効果音再生システム
### 導入
- InputSystemの導入とInputManagerとの共存設定
- UniRx導入
- Dotween導入
- TestFramework導入
